# CodeNest

CodeNest is a multi-page web application for creating, storing and managing project repositories.

## Main features

- Multi-page HTML structure
- Login and logout system
- Project repository CRUD
- User profile page
- Skills and activity blocks
- Contact form
- Admin panel for contact requests
- Search, filters and sorting
- LocalStorage for demo data
- GitHub Pages deployment

## Technologies

- HTML
- CSS
- JavaScript
- LocalStorage
- GitHub Pages

## Pages

- `index.html` — main repository page
- `login.html` — login page
- `profile.html` — user profile
- `project.html` — project terminal/editor
- `contacts.html` — contact form
- `admin.html` — admin panel

## Project purpose

The project demonstrates a web application for managing programming projects with a GitHub-like interface.
